# Bhutan Luxe — Documentation Index

**Project:** Bhutan Luxe / Bhutan Luxury Travel (domain TBD; brand contact `Rare.Bhutan@bhutan-luxe.com`)
**Stack:** Next.js on Vercel (planned) · static single-page scroll · external form endpoint (Lark / Airtable / Resend)
**Owner:** Dave Hajdu
**Last updated:** 2026-04-29

Lead-capture marketing site for a high-end curated travel offer. Sells bespoke Bhutan group journeys to high-net-worth individuals, primarily in Texas, at $8K–$35K+ per person via three tiers (Luxe, Boutique-Luxe, Ultra-Luxe). The site's job is to create desire, present tiers, and capture qualified inquiries — fulfilment is handled by an affiliate partner on the ground in Bhutan.

**Project state (2026-04-29): pre-build.** The repo contains empty HTML scaffolds (`website/index.html`, `about.html`, `readings.html`), empty content folders, and a comprehensive brand style guide. No package.json, no real code, no live deployment yet. This documentation captures the brand intent and the planned build so the first sprint can start cleanly.
